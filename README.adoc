= TorchTest

A Simple Testing Framework for PyTorch, inspired by [mltest](https://github.com/Thenerdstation/mltest/blob/master/mltest/mltest.py) 


== Tests

* Variables Change \/ Don't Change

* Logits Range

* Input Dependencies

* NaN and Inf Tensors
